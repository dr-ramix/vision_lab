# Project Documentation

This folder contains practical documentation for the `vision_lab` FER project.

## Documents

- `project_structure.md`: repository layout and what each major folder does.
- `pre_processing.md`: dataset download and preprocessing pipeline.
- `training.md`: training entry point, key settings, and output artifacts.
- `inference.md`: pretrained-weight workflow and inference API.
- `operations.md`: SLURM jobs, evaluation/testing scripts, and deliverables.

## Recommended Reading Order

1. `project_structure.md`
2. `pre_processing.md`
3. `training.md`
4. `inference.md`
5. `operations.md`
